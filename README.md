# Snake-Game-Simplificado
Meu primeiro jogo desenvolvido com Python + Pygame, recriando o clássico Snake com sistema de movimento em grid, crescimento da cobra e tela de Game Over.
Este projeto representa minha evolução prática em lógica de programação, controle de eventos, estruturas de dados e construção de loops de jogo.

#Controles
| Tecla  | Ação                     |
| ------ | ------------------------ |
| W      | Mover para cima          |
| S      | Mover para baixo         |
| A      | Mover para esquerda      |
| D      | Mover para direita       |
| ESPAÇO | Reiniciar após Game Over |



##Como Executar o Jogo

pip install pygame

##Executar o arquivo

python snake.py


##Funcionalidades Atuais

Movimento da cobra em grid alinhado
Crescimento ao comer a comida
Geração aleatória de comida na tela
Colisão com o próprio corpo
Colisão com as bordas da tela
Tela de Game Over
Reinício do jogo ao pressionar a tecla ESPAÇO
Conceitos de Programação Utilizados
Este projeto aplica fundamentos essenciais de desenvolvimento de jogos:
Loop principal de jogo
Controle de FPS com pygame.time.Clock()
Captura de eventos do teclado
Estrutura de dados lista para o corpo da cobra
Detecção de colisão por coordenadas
Renderização de elementos na tela
Controle de estados do jogo (game_over)
